Kanad Support

Kanad support is configured to use Shopify's built-in contact system instead of an external form provider.

GitHub Pages cannot process Shopify Liquid forms. The live support form must be hosted on a Shopify page that uses the theme's support template.

Live support URL

https://kanaddev.myshopify.com/pages/contact

Native Shopify support flow

1. In Shopify admin, create a page named Support.

2. Assign the page.support template to that page.

3. If the custom template is not yet uploaded to Shopify, use the existing contact template.

4. Publish the page and use its public store URL as your support form URL.

5. The current live store contact page is the URL above.

6. Support emails will be sent to the store admin email through Shopify's built-in contact handling.

Implemented Liquid form

This repository now includes a Shopify-native support form in the theme. It uses Shopify's contact handling and does not require Formspree or another external backend.

{% form 'contact' %}
  <input type="text" name="contact[name]" required>
  <input type="email" name="contact[email]" required>
  <textarea name="contact[body]" required></textarea>
  <button type="submit">Send</button>
{% endform %}
